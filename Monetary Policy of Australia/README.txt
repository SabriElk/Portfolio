This project is based on Clarida-MonetaryPolicyRules paper published in 2000.
This paper provide a comprehensive model to estimate the relationship of interest rate and inflation using lag. Output gap is used as control variables.

This project is therefor a replication of the method applied to a specific monetary Area. The country studies is Australia and the monetary policy driven by Australia's central bank.

The folder is structured as follow:

- Code: This folder contains all code to estimate the model, cleaning data and building necessary graphs
- Data: Contain all necessary data to estimate the model and assess the analysis

And last the paper I based my analysis on.
